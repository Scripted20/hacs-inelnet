# InelNET Blinds Control

Control your InelNET blinds, roller shutters, and awnings from Home Assistant.

## Features

- Open, close, stop, and set position for all your blinds
- Position tracking based on configurable travel time
- Support for up to 32 channels
- Facade orientation tracking for solar automations
- Multi-language support (English, Romanian)

## Quick Start

1. Install via HACS
2. Add the integration in Home Assistant
3. Enter your InelNET IP address
4. Configure your blinds (channel, name, travel time)
5. Done! Your blinds are now controllable from Home Assistant

## Compatibility

Works with all InelNET Internet central control units that support HTTP control.
